# 生日祝福网站

一个精美的生日祝福网站，包含多页祝福内容、动画效果和音乐播放功能。

## 功能特色

- ✨ 精美的飘雪背景动画
- 🎂 多页祝福内容（文字、图片、弹窗、气球礼花）
- 🎵 集成音乐播放器插件
- 📱 响应式设计，支持移动端
- 🎯 手动控制页面切换

## 访问地址

部署后可通过以下地址访问：
`https://您的用户名.github.io/生日网站`

## 本地运行

1. 下载所有文件
2. 用浏览器打开 `index.html`
3. 点击"开启"按钮开始体验

## 文件结构

```
生日网站/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 功能脚本
├── 图片/               # 图片资源
├── 音乐/               # 音乐文件
└── README.md          # 说明文档
```
